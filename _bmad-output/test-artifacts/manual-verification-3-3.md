## AC#1: 删除会话

- [ ] **右键菜单存在**：右键点击 Sidebar 中任意会话 → 出现上下文菜单，包含"重命名"和"删除"选项
- [ ] **删除确认弹窗**：点击"删除" → 弹出 alert，显示"确定要删除「{会话标题}」吗？此操作不可撤销。"
- [ ] **确认删除**：点击"删除" → 会话从列表消失，关联事件级联删除
- [ ] **取消删除**：点击"取消" → 会话仍在列表中
- [ ] **删除最后一个会话**：删除唯一会话后 → 显示空状态（"点击 + 创建新会话"）
- [ ] **删除后自动选中**：删除当前选中会话后 → 自动选中列表中第一个剩余会话

## AC#2: 重命名会话

- [ ] **进入重命名模式**：右键点击 → 选择"重命名" → 标题变为可编辑 TextField，自动聚焦，预填当前标题
- [ ] **Enter 提交**：输入新名称，按 Enter → 标题更新，退出编辑模式
- [ ] **Escape 取消**：按 Escape → 恢复原标题，退出编辑模式
- [ ] **空文本保护**：清空所有文本后按 Enter → 标题不变（空文本不提交）
- [ ] **重命名后排到顶部**：重命名一个旧会话 → 该会话移到列表顶部（updatedAt 排序）
- [ ] **重命名持久化**：重命名后切换到其他会话再切回 → 新标题保留

## AC#3: Agent 执行中追加消息

- [ ] **运行时可输入**：Agent 正在回答时 → 输入框不被禁用，可以继续输入
- [ ] **运行时可发送**：输入文本后点击发送按钮 → 消息追加到 Timeline，不中断当前回答
- [ ] **排队处理**：连续发送 2 条追加消息 → 第一条回答完成后 Agent 自动处理第二条
- [ ] **发送按钮可见**：Agent 运行时 → 发送按钮（蓝色上箭头）和停止按钮（红色方块）同时可见
- [ ] **停止按钮功能**：点击停止按钮 → 当前任务取消，追加消息也取消

## AC#4: Enter 发送 / Shift+Enter 换行

- [ ] **Enter 发送**：输入文本后按 Enter → 消息发送，输入框清空
- [ ] **Shift+Enter 换行**：按 Shift+Enter → 插入换行，不发送消息
- [ ] **Option+Enter 换行**：按 Option+Enter → 同样插入换行
- [ ] **多行文本发送**：输入多行文本后按 Enter → 完整多行文本作为一条消息发送
- [ ] **中文 IME 安全**：使用中文输入法输入，按 Enter 确认候选字 → 不误触发发送
