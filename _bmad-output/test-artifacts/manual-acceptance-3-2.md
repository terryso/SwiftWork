## AC#2: 规则删除，后续同类操作重新要求审批

**Given** 用户查看权限规则列表
**When** 点击某条规则并选择删除
**Then** 规则从列表和 SwiftData 中移除，后续同类操作将重新要求审批

### 单条删除

- [ ] 每行规则支持左滑删除操作
- [ ] 滑动删除单条规则时弹出确认对话框
- [ ] 确认对话框标题为"删除权限规则"
- [ ] 确认对话框显示规则工具类型名称
- [ ] 点击"取消"后规则保留，不做任何操作
- [ ] 点击"删除"后规则从列表中消失
- [ ] 删除后规则计数更新（如"权限规则（3 条）"→"权限规则（2 条）"）

### 批量删除

- [ ] 多选删除（如有）可正常工作
- [ ] 批量删除后所有相关规则被移除

### 删除后行为验证

- [ ] 删除一条"始终允许 Bash"的规则后，手动审批模式下再次执行 Bash 命令时重新弹出审批弹窗
- [ ] 删除所有规则后列表显示空状态提示
- [ ] 删除规则不影响其他已有规则

### 边界情况

- [ ] 空列表中执行删除操作不会崩溃
- [ ] 快速连续点击删除不会导致重复删除或崩溃

## AC#3: 全局权限模式切换，立即生效 (FR26)

**Given** 用户打开设置页面
**When** 查看全局权限模式选项
**Then** 可选择"自动批准"、"手动审批"、"全部拒绝"模式
**And** 模式选择立即生效，影响后续所有工具调用的评估逻辑

### 模式选择器

- [ ] 全局权限模式区域显示分段选择器（Segmented Picker）
- [ ] 三个选项：自动批准、手动审批、全部拒绝
- [ ] 当前选中模式高亮显示
- [ ] 每个模式下方显示简要说明文字：
  - 自动批准："所有工具调用自动批准，无需用户确认"
  - 手动审批："每次工具调用都需要用户审批"
  - 全部拒绝："拒绝所有工具调用"

### 模式切换立即生效

- [ ] 切换到"手动审批"后，Agent 执行工具调用时弹出审批弹窗
- [ ] 切换到"自动批准"后，Agent 执行工具调用时不弹窗，直接执行
- [ ] 切换到"全部拒绝"后，Agent 执行工具调用被直接拒绝
- [ ] 模式切换无需重启 App 即刻生效

### 模式持久化

- [ ] 切换模式后关闭设置页面，模式保持
- [ ] 重启 App 后全局模式恢复为上次选择的值
- [ ] 多次切换后保存的是最后一次选择的值

### 模式切换不影响 Agent 执行

- [ ] 打开设置页面时 Agent 不会中断正在执行的任务
- [ ] 关闭设置页面后 Agent 继续正常工作
