## AC #1: PlanView 显示步骤列表（FR34）

**目标：** 验证 Agent 进入/退出计划模式时，PlanView 正确渲染步骤列表

- [ ] **1.1 EnterPlanMode 事件**
  - 步骤：发送消息触发 Agent 进入计划模式（如"先规划再执行"类提示词）
  - 预期：Timeline 中出现青色（teal）左侧竖条的计划卡片，显示"进入计划模式"

- [ ] **1.2 ExitPlanMode 事件（含编号列表）**
  - 步骤：Agent 完成计划后退出计划模式
  - 预期：计划卡片显示摘要（如"执行计划 (3 步骤, 0 完成)"），点击展开后显示编号步骤列表

- [ ] **1.3 ExitPlanMode 事件（含 Markdown 列表）**
  - 步骤：使用不同提示词让 Agent 生成 `- step` 或 `* step` 格式的计划
  - 预期：步骤列表正确解析并显示

- [ ] **1.4 折叠/展开交互**
  - 步骤：点击计划卡片标题行，展开步骤列表；再次点击折叠
  - 预期：展开/折叠有平滑动画，折叠时显示摘要和步骤计数

- [ ] **1.5 无结构化计划的降级**
  - 步骤：Agent 输出无法解析为列表的自由文本计划
  - 预期：展开后显示原始计划文本（而非空白）
